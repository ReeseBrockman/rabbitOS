# rabbitOS 🐇

A macOS menu bar app that drops a animated bunny down from the notch area.

## Features
- Slides down from the notch on hover
- Day/night mode based on time of day
- Twinkling stars at night with shooting stars
- Drifting clouds during the day
- Eyes track your mouse position
- Auto dismisses after 5 seconds

## Built With
- Swift
- SwiftUI
- macOS 15

## Installation
Clone the repo and open `rabbitOs.xcodeproj` in Xcode.
